# --description--

이제 웹폼을 만들어보겠습니다.

`input` 요소는 사용자로부터 입력을 받는 편리한 방법입니다.

다음과 같이 글자 입력을 만들 수 있습니다.

```html
<input type="text">
```

`input` 요소는 스스로 여닫는 태그임을 알아두시오.

# --instructions--

목록 하위에 `text` 유형의 `input` 요소를 만드시오.

# --hints--

어플리케이션은 `text` 유형의 `input` 요소를 가져야 합니다.

```js
assert($('input[type=text]').length > 0);
```
