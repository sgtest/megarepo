## --text--

`Console.Write` と `Console.WriteLine` の違いは何ですか？

## --answers--

`Console.Write` は出力を新しい行に表示する

---

`Console.WriteLine` は出力を新しい行に表示する

---

`Console.WriteLine` は出力の後に新しい行を追加する
