## --text--

メソッドはいつ例外をスローすべきですか？

## --answers--

メソッドは、意図した目的を完了できる場合に例外をスローすべきである。

---

メソッドは、意図した目的を完了できない場合に例外をスローすべきである。

---

メソッドは、例外の種類が意図した目的に一致する場合に例外をスローすべきである。
