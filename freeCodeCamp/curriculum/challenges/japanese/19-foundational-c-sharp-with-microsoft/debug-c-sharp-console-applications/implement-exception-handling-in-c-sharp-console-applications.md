## --text--

`ArrayTypeMismatchException` 例外は、どのような場合にスローされますか？

## --answers--

ある型の値を別の型の配列に格納しようとした場合、`ArrayTypeMismatchException` 例外がスローされる。

---

ある型の値が null のメンバーにアクセスしようとした場合、`ArrayTypeMismatchException` 例外がスローされる。

---

配列の範囲外にインデックスを作成しようとした場合、`ArrayTypeMismatchException` 例外がスローされる。
