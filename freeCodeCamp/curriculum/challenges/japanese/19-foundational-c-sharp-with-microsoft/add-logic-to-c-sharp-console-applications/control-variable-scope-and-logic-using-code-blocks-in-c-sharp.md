## --text--

`if` ステートメントと関連付けられたコードブロックの中括弧の追加または削除について、正しい文は次のうちどれですか？

## --answers--

`else if` および `else` ステートメントのコードブロックから中括弧を削除することはできない。

---

`if-elseif-else` のコードブロックから中括弧を削除する場合、スペースも削除しなければならない。

---

常に、読みやすさを向上させるスタイルを選択する。
