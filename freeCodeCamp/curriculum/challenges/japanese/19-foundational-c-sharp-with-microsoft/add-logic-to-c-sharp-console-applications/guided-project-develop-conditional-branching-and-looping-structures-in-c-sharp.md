## --text--

`if-elseif-else` 構文より `switch-case` 構文を使う方が適切なのは、どのような場合ですか？

## --answers--

`else if` コードブロックが 2、3 個より多く必要になる場合、`switch-case` 構文が適切である。

---

`case` パターンの数が少ない場合、`switch-case` 構文が適切である。

---

選択ステートメントがループ内にある場合、`switch-case` 構文が適切である。
