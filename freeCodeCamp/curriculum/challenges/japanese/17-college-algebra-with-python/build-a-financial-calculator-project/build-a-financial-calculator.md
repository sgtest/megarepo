# --instructions--

このチャレンジでは、入力を受け取り、次の処理を実行する財務計算機を Python を使用して作成する必要があります。

- 毎月または連続的に増加する年金を計算する
- 毎月の住宅ローン支払い額を計算する
- 退職後の投資残高を見積もる
- 指定した利率で金額が 2 倍になるまでの期間を調べる
- 対数方程式を解く
- 科学的記数法への変換 (またはその逆) を行う
