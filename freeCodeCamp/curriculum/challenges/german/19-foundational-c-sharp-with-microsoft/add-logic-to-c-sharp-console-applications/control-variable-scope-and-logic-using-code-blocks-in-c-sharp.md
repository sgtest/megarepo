## --text--

Welche der folgenden Aussagen über das Anzeigen/Entfernen von geschweiften Klammern bei Codeblöcken, die mit einer `if`-Anweisung verbunden sind, ist richtig?

## --answers--

Die geschweiften Klammern können nicht aus dem Code-Block für `else if` und `else`-Anweisungen entfernt werden.

---

If the curly braces are removed from the code blocks of an `if-elseif-else`, the white space must also be removed.

---

Wähle immer einen Stil, der die Lesbarkeit verbessert.
