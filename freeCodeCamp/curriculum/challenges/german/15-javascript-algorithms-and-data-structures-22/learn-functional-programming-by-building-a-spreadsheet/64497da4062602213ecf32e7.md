# --description--

Da das `change`-Ereignis auf einem `input`-Element ausgelöst wird, hat das Element eine `value`-Eigenschaft, die den aktuellen Wert der Eingabe darstellt.

Weise die `value`-Eigenschaft von `element` einer neuen Variablen namens `value` zu um´und verwende `.replace()`, um alle Leerzeichen zu entfernen.
