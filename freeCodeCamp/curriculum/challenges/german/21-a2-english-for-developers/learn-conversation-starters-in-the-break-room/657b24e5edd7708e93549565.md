# --description--

Das Präsent Perfekt ist eine Zeitform, die verwendet wird, um eine Verbindung zwischen der Gegenwart und der Vergangenheit herzustellen. Der Zeitpunkt der Handlung liegt vor der Gegenwart, ist aber nicht festgelegt, und die Person, die sie benutzt, ist in der Regel mehr am Ergebnis als an der Handlung selbst interessiert. Zum Beispiel:

`I played this game yesterday.` - Das ist die einfache Vergangenheitsform. Es besagt, dass die Aktion das Spiel zu spielen zu einer bestimmten Zeit in der Vergangenheit passiert ist, welche gestern ist.

`I have played this game before.` - This is in the present perfect tense. Es besagt, dass die Aktion das Spiel zu spielen zu einer unspezifischen Zeit in der Vergangenheit stattgefunden hat.

`I’ve played this game before.` - This sentence is also in the present perfect tense, using the contraction `I've` for `I have`. Wie im vorigen Beispiel, wird besagt, dass die Handlung das Spiel zu spielen, zu einer unbestimmten Zeit in der Vergangenheit passierte.
