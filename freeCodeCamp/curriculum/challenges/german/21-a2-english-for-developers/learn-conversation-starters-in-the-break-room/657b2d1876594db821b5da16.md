# --description--

In this part of the dialogue, Sarah suggests playing `Gartic` on the next game night. Sie verwendet die Worte `Maybe` und `we can`, um die Möglichkeit dieses Ereignisses auszudrücken.

`Maybe` ist ein Wort, das du benutzt, wenn du eine Möglichkeit vorschlägst, aber dir nicht sicher bist, ob diese Möglichkeit passieren wird oder nicht. Es zeigt, dass du keinen endgültigen Plan hast, aber offen für die Idee bist. For example: if someone asks you if you want to go to the movies, you can say `Maybe` if you are not sure if you want to go or if you are available.

`We can` wird verwendet, um eine Idee vorzuschlagen oder etwas höflich anzuregen. Es zeigt, dass die Gesprächsteilnehmer die Fähigkeit oder die Möglichkeit haben, etwas zu tun. Zum Beispiel, wenn du vorschlagen möchtest ein Spiel zu spielen, kannst du sagen `We can play a game` um die Idee anzubieten.

The difference between `maybe` and `we can` is that `maybe` suggests a possibility without making a definite plan, while `we can` suggests an idea or proposal in a polite way.
