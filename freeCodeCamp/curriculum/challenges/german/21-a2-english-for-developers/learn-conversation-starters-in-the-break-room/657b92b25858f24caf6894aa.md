# --description--

The expression `only if` is used to set a condition for something to happen. Es ist als würde man sagen `this will happen if that happens.` Es ist als würde man eine Vereinbarung treffen, bei der eine Aktion von einer anderen abhängig ist.

Zum Beispiel: wenn dein Freund sagt, `I'll go out only if it doesn't rain,` bedeutet es, er wird raus gehen aber das Wetter muss gut sein - kein Regen.
