# --description--

Визначимо $Co(n)$ як максимально можливу суму множини взаємно простих елементів з $\\{1, 2, \ldots, n\\}$. Наприклад, $Co(10)$ дорівнює 30 і досягає цього максимуму в підмножині $\\{1, 5, 7, 8, 9\\}$.

Дано, що $Co(30) = 193$ та $Co(100) = 1356$.

Знайдіть $Co(200\\,000)$.
