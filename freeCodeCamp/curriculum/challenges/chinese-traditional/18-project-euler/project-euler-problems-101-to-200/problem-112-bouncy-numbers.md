# --description--

Working from left-to-right if no digit is exceeded by the digit to its left it is called an increasing number; for example, 134468.

同樣，如果右邊的數字沒有超過任何數字，則稱爲遞減數；例如，66420。

我們將既不增加也不減少的正整數稱爲“有彈性”數；例如，155349。

顯然，不可能有低於一百的任何有彈性的數字，但低於一千 (525) 的數字中只有一半以上是有彈性的。 事實上，有彈性數字的比例首先達到 50% 的最小數字是 538。

令人驚訝的是，有彈性的數字變得越來越普遍，當我們達到 21780 時，有彈性的數字的比例等於 90%。

找出有彈性數的比例恰好爲 99% 的最小數。
