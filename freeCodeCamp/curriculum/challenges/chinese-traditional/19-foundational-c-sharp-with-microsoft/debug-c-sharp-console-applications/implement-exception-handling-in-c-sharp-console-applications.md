## --text--

何時拋出 `ArrayTypeMismatchException` 異常？

## --answers--

當嘗試將一種類型的值存儲在另一種類型的值的數組中時，會拋出 `ArrayTypeMismatchException` 異常。

---

當試圖訪問值爲 null 的成員類型時，會拋出 `ArrayTypeMismatchException` 異常。

---

當嘗試索引數組超出數組邊界時，會拋出 `ArrayTypeMismatchException` 異常。
