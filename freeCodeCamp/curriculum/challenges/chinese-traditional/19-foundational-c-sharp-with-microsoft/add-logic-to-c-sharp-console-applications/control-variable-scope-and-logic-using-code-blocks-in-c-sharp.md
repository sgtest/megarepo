## --text--

關於顯示/刪除與 `if` 語句關聯的代碼塊的大括號，以下哪項陳述是正確的？

## --answers--

不能從 `else if` 和 `else` 語句的代碼塊中刪除大括號。

---

如果從 `if-elseif-else` 代碼塊中刪除大括號，則也必須刪除空格。
