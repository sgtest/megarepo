## --text--

要對字符串數組進行排序，爲什麼在排序前對每個數組元素使用 `String.Trim()` 方法很重要？

## --answers--

只有元素被 `String.Trim()` 處理後的字符串數組才能排序。

---

`String.Trim()` 移除了會排在數字和字母之前的前導空格。

---

對每個元素使用 `String.Trim()` 的數組排序速度顯著加快。
