# --description--

Consider the triangles with integer sides $a$, $b$ and $c$ with $a ≤ b ≤ c$.

如果$gcd(a,b,c) = 1$，则整数边三角形$(a,b,c)$称为基元。

存在多少个原始整数边三角形，其周长不超过10 00,0 000？
