# --description--

<dfn>`while`</dfn> 循环将不断运行，直到指定的 `condition` 不再成立。 它的语法如下：

```js
while (condition) {
  logic;
}
```

使用该语法声明一个以 `continueLoop` 为条件的 `while` 循环。 主体应该是空的。
