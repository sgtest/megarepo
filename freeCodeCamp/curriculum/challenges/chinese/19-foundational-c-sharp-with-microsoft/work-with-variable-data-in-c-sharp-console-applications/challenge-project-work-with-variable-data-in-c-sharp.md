## --text--

要对字符串数组进行排序，为什么在排序前对每个数组元素使用 `String.Trim()` 方法很重要？

## --answers--

只有元素被 `String.Trim()` 处理后的字符串数组才能排序。

---

`String.Trim()` 移除了会排在数字和字母之前的前导空格。

---

对每个元素使用 `String.Trim()` 的数组排序速度显著加快。
