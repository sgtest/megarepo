关于显示/删除与 `if` 语句关联的代码块的大括号，以下哪项陈述是正确的？

## --answers--

不能从 `else if` 和 `else` 语句的代码块中删除大括号。

---

如果从 `if-elseif-else` 代码块中删除大括号，则也必须删除空格。

---

始终选择能够提高可读性的代码风格。
