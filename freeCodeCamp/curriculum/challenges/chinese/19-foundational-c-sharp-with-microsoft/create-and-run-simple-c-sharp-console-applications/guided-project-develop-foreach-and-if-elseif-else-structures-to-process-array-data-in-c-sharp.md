## --text--

开发者编写一个应用程序，该程序使用 `foreach` 循环来遍历一个包含 20 个元素的数组。 应用程序完成后，开发者了解到，必须更新数组以包含 40 个元素。 应用程序需要检查所有这 40 个数组元素。 下列哪项描述了所需的代码更新？

## --answers--

创建第二个 `foreach` 循环，遍历添加的项目。

---

无需更改 `foreach` 循环。

---

将原始的 `foreach` 循环放在第二个 `foreach` 循环的代码块中，两次迭代。
