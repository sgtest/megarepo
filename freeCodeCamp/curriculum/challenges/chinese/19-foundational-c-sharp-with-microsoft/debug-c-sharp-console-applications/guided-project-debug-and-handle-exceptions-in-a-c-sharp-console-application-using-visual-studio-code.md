## --text--

何时抛出 `ArgumentOutOfRangeException` 异常？

## --answers--

当尝试索引数组超出数组边界时，会抛出 `ArgumentOutOfRangeException` 异常。

---

当参数的值在方法定义的允许值范围之外时，会抛出 `ArgumentOutOfRangeException` 异常。

---

当尝试将一种类型的值存储在另一种类型的值的数组中时，会抛出 `ArgumentOutOfRangeException` 异常。
