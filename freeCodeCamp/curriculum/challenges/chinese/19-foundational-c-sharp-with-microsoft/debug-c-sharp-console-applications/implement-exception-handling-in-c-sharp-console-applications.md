## --text--

何时抛出 `ArrayTypeMismatchException` 异常？

## --answers--

当尝试将一种类型的值存储在另一种类型的值的数组中时，会抛出 `ArrayTypeMismatchException` 异常。

---

当试图访问值为 null 的成员类型时，会抛出 `ArrayTypeMismatchException` 异常。

---

当尝试索引数组超出数组边界时，会抛出 `ArrayTypeMismatchException` 异常。
