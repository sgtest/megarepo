# Cody CLI (experimental)

Cody CLI is an experimental CLI of Cody.

## Install

In the root of the repository, run this:

```
pnpm --filter @sourcegraph/cody-cli run build
cd client/cody-cli
npm install -g .
```

## Local Development

In the root of the repository, run this:

```
pnpm --filter @sourcegraph/cody-cli run start
```
