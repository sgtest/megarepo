# otel-collector

Configuration for running otel-collector in development. `config.yaml` should be mounted to a path specified via the `--config` flag.

Debug pages ("zPages") are available at port 55679 by default - see [Exposed zPages routes](https://github.com/open-telemetry/opentelemetry-collector/blob/main/extension/zpagesextension/README.md#exposed-zpages-routes).
